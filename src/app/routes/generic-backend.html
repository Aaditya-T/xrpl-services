xumm.community Generic Backend


How do I use the XUMM Generic Backend?

The xumm.community Generic Backend has a very simple API with some really cool features. Most of the "hard work" is done automatically by the backend. (Setting your destination address, setting up XUMM push notifications, and so on ...)

XUMM Pay Button

The XUMM Generic Pay Button offers you an easy possibility to add a "XUMM Pay" button into your website. As example: for donations/tips/payments but also for offering exclusive content (after payment)


How to use the XUMM Generic Pay Button [XGPB ;-)]:

You have to be whitelisted for the xumm.community Generic Backend (see steps above). After you are whitelisted:
Add the following script into the <head> section of your website:

<script async src="https://pay.xumm.community/payViaXumm.js"></script>

Now you can add a <div> tag anywhere in your page where you want to have a pay button.

The <div> tag must have the attribute: class="xumm-pay-button":
<div class="xumm-pay-button"></div>

The XUMM Generic Pay Button will be inserted into the <div> element. If you want to change the size of the button, just change the size of the surrounding <div>.
